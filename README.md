This is PIT (a clone of GIT)

STEPS To Use:
  1) Clone this repo : https://github.com/MuhammadQasimHamid/MyGitTest
  2) Open MyGitTest/build/bin
  3) Add the complete absolute path (e.g, C:/Data/MyGitTest/build/bin) to Enviormental Variables.
  4) Open CMD and run 'pit init'
  5) Pit repo initialized, Test & Enjoy.
